
  # E-commerce Gift Shop App

  This is a code bundle for E-commerce Gift Shop App. 
  Ecommerce GiftShop WebApp (In Progress)

A full-stack e-commerce gift shop web application built using React (Vite), Node.js, MongoDB, and Tailwind CSS. The platform allows users to buy and sell gift items, manage products, and request custom orders through in-app messaging. This project focuses on practical full-stack architecture, API integration, and responsive UI design.

🚀 Features

User registration and login authentication

Product listing and browsing system

Buy & sell gift items functionality

Seller dashboard for managing products

Order and user data stored in MongoDB

Responsive UI built with Tailwind CSS

Custom order requests via buyer–seller messaging

REST API integration between frontend and backend

🛠 Tech Stack

Frontend: React (Vite), Tailwind CSS

Backend: Node.js, Express

Database: MongoDB



  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
